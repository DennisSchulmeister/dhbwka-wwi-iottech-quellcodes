Anforderungen an das IoT-Device

Hardware

  • Alle 30 Sekunden sollen Temperatur und Luftfeuchtigkeit gemessen werden.
  • Über einen Hardwaretaster soll die Messung jederzeit manuell angestoßen werden können.
  • Die Messung soll zwei Sekunden dauern, damit sich die Sensorwerte stabilisieren können.
  • Während der Messung soll eine LED blinken, damit man sieht, dass das Device aktiv ist.

Messwerte

  • Jeder Messwert soll mit dem Zeitstempel seiner Messung versehen werden
  • Die gemessenen Werte sollen in einer lokalen SQLite-Datenbank gespeichert werden.
  • Die letzten, gemessenen Werte sollen über eine HTTP REST-Schnittstelle abrufbar sein.

Programmierung

  • Der Quellcode soll modular aufgebaut werden und einfach zu erweitern sein.
  • Die wichtigsten Arbeitsschritte des Programms sollen auf der Konsole protokolliert werden.
