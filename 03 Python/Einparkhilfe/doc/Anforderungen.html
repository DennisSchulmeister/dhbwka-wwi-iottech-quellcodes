Anforderungen an das Device

Es soll eine Einparkhilfe programmiert werden, wie man sie vom Auto her kennt. Das heißt, mit Hilfe eines Ultraschallsensors soll der Abstand zum nächsten Hinderniss gemessen und je nach Abstand ein immer schneller werdender Piepston ausgegeben werden.

Hardware

  • Die Messung soll zunächst inaktiv sein.
  • Durch Drücken eines Hardwarebuttons wird der Sensor aktiviert.
  • Durch erneutes Drücken wird der Sensor wieder deaktiviert.
  • Die Messung soll mehrmals pro Sekunde erfolgen.
  • Die Entfernung soll durch eine LED und einen Piepser signalisiert werden.
  • Das Piepsen soll umso schneller werden, je näher das Hinderniss kommt.

Programmierung

  • Der Quellcode soll modular aufgebaut werden und einfach zu erweitern sein.
  • Die wichtigsten Arbeitsschritte des Programms sollen auf der Konsole protokolliert werden.
  • IoT-Anforderungen müssen nicht erfüllt werden, aber einfach nachrüstbar sein.
  • Optional soll das Programm über eine grafische Oberfläche verfügen.
